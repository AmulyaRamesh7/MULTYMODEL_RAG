# MULTYMODEL_RAG

# Enterprise Multimodal RAG (Text + Vision)

End-to-end Multimodal RAG using:

- Groq LLM
- Groq Vision
- Jina Embeddings v4
- FAISS
- Streamlit

## Setup

1. Clone repo
2. Add .env file:

JINA_API_KEY=your_key
GROQ_API_KEY=your_key

3. Install dependencies:

pip install -r requirements.txt

4. Run:

streamlit run app.py
